My QMK Keyboard Config

 I have a reprogrammable keyboard which can bind anything to anything using the open source QMK firmware. This is the layout I use for my OLKB Planck Rev.6. It allows quick switching between QWERTY, Colemak and Dvorak, allows the user to move the mouse and click with just the keys and overlays different common functions on the same key so you never have to stretch your hands.

Prayer Times Finder

 A program I wrote to grab Islamic prayer times and display them in a terminal. Originally made to be run on a phone with Termux, as an alternative to proper prayer time apps. Written in Racket, because it's such a nice language.
