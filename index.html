My QMK Keyboard Config

 I have a reprogrammable keyboard which can bind any key to any action using the open source QMK firmware. This is the layout I use for my OLKB Planck Rev.6. It allows quick switching between QWERTY, Colemak and Dvorak, allows the user to move the mouse and click with just the keys and overlays different common functions on the same key so you never have to stretch your hands.
