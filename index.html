Embedded Development

I like playing with microcontrollers. I have experience using Arduino boards with their IDEs to make tiny machines do interesting things. My largest project created completely by myself was a TAS tool designed for my university final year project. I used a Beaglebone Black and a Teensy 2.0 to create a device which could pretend to be a keyboard and play back prerecorded inputs, similar to a Pianola.

My second large project is a programmable mechanical keyboard keymap. I write code in C containing information on what I want each key to do, and any other code I want the keyboard to execute, and then flash it directly to the microcontrollers powering the keyboard. This allows me to adjust everything about my keyboard to suit my needs, to improve ergonomics and add more functionality
